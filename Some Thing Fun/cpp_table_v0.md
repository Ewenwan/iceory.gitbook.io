# Multi-Function Table
---

GitHub地址：https://github.com/ICEORY/Learning-C-Plus-Plus/tree/master/multifunction-table

## 功能说明

创建一个表格，实现创建多个列，对每一列的数据类型以及列名进行定义，完成之后，可以对数据进行自动排序。
